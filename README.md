# Actions Microservice

Микросервис для рекомендации действий пользователю на основе контекста окружающего мира (погода, время суток, сезон, ближайшие места).

## Функциональность

- Получение списка доступных действий (прогулка, сон, просмотр фильма, прослушивание музыки, посещение места)
- Генерация персонализированных рекомендаций на основе:
    - Погодных условий (температура, осадки)
    - Времени суток (утро, день, вечер, ночь)
    - Сезона (зима, весна, лето, осень)
    - Ближайших развлекательных мест
- Сохранение истории рекомендаций пользователя
- Умный подбор фильмов и музыки из локальной базы (JSON)
- Swagger-документация API
